
## HTML 글자색 바꾸기 

### 코드
``` html
<!DOCTYPE html>
<html>
	<body>
		<h1 style="color:blue;">제목(파랑색)</h1>
	</body>
</html>
```
### 결과

<!DOCTYPE html>
<html>
	<body>
		<h1 style="color:blue;">제목(파랑색)</h1>
	</body>
</html>

``` html
<!DOCTYPE html>
<html>
	<head>
		<style>
			body {background-color:lightgrey}
			h1 {color:blue}
			p {color:red}
		</style>
	</head>
	<body>
		<h1>제목(파란색)</h1>
		<p>문장(빨간색)</p>
	</body>
</html>
```
